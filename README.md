# AI Phishing Website Detection 

Machine Learning powered phishing detection system that analyzes website URLs and webpage behavior to detect malicious websites.

## Features

* URL based phishing detection
* Webpage structure analysis
* Machine Learning prediction
* Risk score dashboard
* Streamlit UI

## Tech Stack

* Python
* Scikit-learn
* AutoML Model
* Streamlit
* BeautifulSoup
* Requests

## Run Project

```bash
pip install -r requirements.txt
streamlit run app.py
```
